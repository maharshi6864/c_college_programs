Q-3 Explain two functionality of “mv” command with example?

Using mv command we can rename the directory as well file name AND also we can change the file location
or folder location to another location

command:
  //To rename a file
  mv <FILE NAME (OR) FOLDER NAME> <NEW NAME>
  //To move the file location
  //remember while changing the file location a dot is essential and if not the full location is required.
  mv <FILE NAME (OR) FOLDER NAME> <target location>
